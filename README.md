### Introduction
A simple simulation pakage for depth camera images

### Install
cd catkin_ws/src
git clone

### Build
cd catkin_ws
catkin_make
source devel/setup.bash

### Launch
roslaunch depth_camera_sim depth_camera_sim.launch
